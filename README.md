# Tic-Tac-Toe (Vanilla JS)

A simple, interactive Tic-Tac-Toe game built as part of my JavaScript learning journey this month. 

## 🎯 Learning Objectives
This project was focused on mastering the basics of web development without using any frameworks:
* **DOM Manipulation:** Handling clicks and updating the UI dynamically.
* **Game Logic:** Implementing win-condition checks using arrays.
* **State Management:** Keeping track of current players (X vs O) and draw states.

## 🚀 Features
* Clean and responsive grid UI.
* Real-time "Turn" indicator.
* Automatic Win/Draw detection with a reset button.

## 🛠️ Tech Stack
* **HTML5** for structure.
* **CSS3** for the grid layout and styling.
* **JavaScript (ES6)** for the game engine.
